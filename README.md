# shebavideo player

webapp video player with built in keyboard control picture in picture mode autoplay and speed extension up to 5x speed

### formats supported
                "avi", "divx", "flv", " mkv", "mov", "mp4", "mpeg", "mpg", "ogm", "ogv", "ogx", "rm", "rmvb", "smil", "webm", "wmv", "xvid"
